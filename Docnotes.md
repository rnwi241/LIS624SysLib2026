### Documentation Notes for New Requirements

```
cd LIS624SysLib2026
```

## Step 2. Create file

To create markdown files in nano indicate the name of the file first

Example: nano Apache.md

## Step 3. Create markdown language

See assignment specifications.

Use textual and markup language to create .md files for GitHub repo. 

## Step 4. Exit and Save

Use ctrl = X to exit, Y to save.

## Step 5. Git Add

For file specific designations use file name.

```

```

To add all files

```
git add .
```

git commit -m “cleaned up mark down”	EXAMPLE
